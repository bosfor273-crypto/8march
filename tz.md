Ты — Senior Frontend Developer. Твоя задача — разработать мобильный лендинг к 8 Марта на стеке: Vue 3 (Composition API), TypeScript, Vite.

Проект должен быть готов к запуску через `npm install` и `npm run dev`.

### 1. Технический стек и настройки

- **Фреймворк:** Vue 3 (используй `<script setup lang="ts">`).
- **Сборщик:** Vite.
- **Стили:** Scoped CSS внутри компонентов или Tailwind CSS (на твой выбор, но код должен быть сразу рабочим). Предпочтение отдаю чистому CSS с переменными для легкой кастомизации.
- **Анимация:** Используй легкую библиотеку `canvas-confetti` для фейерверков (укажи установку в инструкции).
- **Типизация:** Строгая TypeScript типизация для всех данных и событий.

### 2. Структура данных (TypeScript)

- Создай интерфейс `Poem` (поля: `id`, `text`, `author?`).
- Создай массив из 7 поздравительных стихов на русском языке.
- Реализуй логику случайного выбора стиха при монтировании компонента (`onMounted`)
- Реализуй наличие в url токена для определения имени пользователя из настроечного json, обратившись к пользователю по Имени подраи ему стихотворение.

### 3. Функциональные требования (App.vue)

- **Состояние (State):**
  - `currentPoem`: выбранный стих.
  - `isButtonVisible`: видимость кнопки (true/false).
  - `isBouquetVisible`: видимость букета (true/false).
  - `isCelebrating`: статус запуска фейерверков.
- **Логика кнопки:**
  - Кнопка с текстом "Тут твой букет".
  - При клике:
    1. Кнопка скрывается.
    2. Запускается функция фейерверков (duration ~3 секунды).
    3. После завершения фейерверков (используй `setTimeout` или Promise) плавно проявляется букет (CSS transition opacity).
- **Букет:** Используй качественное изображение SVG-иллюстрацию. Найди в интернете 14 разных изображений букетов с весенними цветами и используй их в проекте.

### 4. Дизайн и UI/UX (Mobile First)

- **Адаптивность:** Верстка должна идеально смотреться на мобильных устройствах (viewport meta tag настроен).
- **Цвета:** Пастельная гамма (розовый, белый, мягкий зеленый).
- **Типографика:** Подключи Google Fonts (например, 'Montserrat' для текста, 'Playfair Display' для заголовков).
- **Анимации:**
  - Плавное появление стиха при загрузке (`fade-in`).
  - Плавное появление букета после салюта.
  - Кнопка должна иметь приятный эффект нажатия (active state).
