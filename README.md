# 🌸 Поздравление к 8 Марта

Мобильный лендинг для поздравления с Международным женским днем.

## ✨ Особенности
- **Персонализация:** Имя получателя подтягивается из URL через токен.
- **Случайный выбор:** При каждой загрузке выбирается один из 7 праздничных стихов.
- **Интерактив:** Праздничный салют (конфетти) перед вручением букета.
- **Галерея:** 14 различных встроенных SVG-иллюстраций весенних букетов.
- **Адаптивность:** Дизайн оптимизирован под мобильные устройства (Mobile First).

## 🛠 Технологии
- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript** (строгая типизация)
- **Vite** (быстрая сборка)
- **Canvas-confetti** (анимация салюта)
- **Google Fonts** (Montserrat, Playfair Display)

## 🚀 Запуск проекта

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Запуск в режиме разработки:**
   ```bash
   npm run dev
   ```

3. **Просмотр:**
   Откройте в браузере ссылку с токеном для проверки персонализации:
   `http://localhost:5173/?token=user1`

## 📝 Настройка имен
Список доступных токенов и имен находится в файле `public/users.json`. Вы можете добавить свои токены:
```json
{
  "my-token": "Имя Получателя"
}
```
После чего ссылка будет: `?token=my-token`.

## 🎨 Дизайн
Использована нежная пастельная гамма:
- Розовый (`#ffb7c5`)
- Мягкий белый (`#fff5f7`)
- Весенний зеленый (`#e2f0cb`)
